🩸 Blood Donation Management System

Built a full-stack web application to connect blood donors with those in need, streamlining the donation process and making it more accessible.

🔧 Tech Stack:
• Backend: Python, Django, Django REST Framework
• Database: MySQL
• Frontend: HTML, CSS, JavaScript (Django Templates)

✨ Key Features:
• Donor registration and profile management
• Blood group search and filtering
• Admin dashboard for managing donations and requests
• REST API integration

This project was built to address the real-world challenge of blood availability during emergencies by digitizing the donor-recipient connection process.
